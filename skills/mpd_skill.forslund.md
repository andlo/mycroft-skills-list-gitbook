### _mpd_skill.forslund_  
## Description:  
By default the mpd skill tries to connect to localhost on port 6600 if this is not desirable edit the mycroft.ini file
[MPDSkill]
mpd_url=YOUR_URL
mpd_port=YOUR_PORT
