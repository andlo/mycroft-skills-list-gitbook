description: Set alarms with Mycroft
---
Set a single use alarm, daily alarms, or weekly recurring alarms.

You can choose from five different alarm sounds:
* Constant Beep
* Four rapid beeps
* Escalating beeps
* Alarm bell
* Gentle chimes.
